# Sell-Cohort Terminology — HARD RULE

## Never say
- "cheetah cohort"
- "cheetah doomsday"
- "cheetahs"
- any animal metaphor for sellers

- **"7% monthly sell-off cohort"** (canonical)
- "100%-dump cohort" (acceptable variant)
- "monthly full-sellers" (acceptable variant)

"Cheetah" is a voice-to-text artifact for **Tschida** — Joseph's co-founder Michael Tschida (CFO/CRO).
Using "cheetah" for sell pressure is confusing and disrespectful to the co-founder who identified the broken-flywheel gap.

## How to apply
- Rename any existing file, variable, doc, or memory containing "cheetah" → use neutral sell-cohort language
- `cheetah-doomsday-rebuttal.md` should be renamed/rewritten as `sell-pressure-rebuttal.md` next time it's touched
- Tschida = the person. The 7% sell cohort = the risk. Never the same word.
